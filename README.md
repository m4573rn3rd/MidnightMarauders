# MidnightMarauders

## About the Addon

MidnightMarauders is a comprehensive World of Warcraft addon for tracking personal and guild statistics. It provides valuable telemetry on combat performance, guild contributions, questing, and market data. The addon is designed to be lightweight and easy to use, with a focus on providing useful information to the player.

## Features

-   **Combat Tracker:**
    -   Tracks damage done by you and your group members during boss encounters.
    -   Automatically starts and stops tracking on boss engages and kills.
    -   Reports top DPS in instance, raid, or party chat after a boss victory.
    -   Takes a screenshot automatically when a boss is defeated.
-   **Quest Tracker:**
    -   Provides a movable HUD to display the number of quests completed today and in total.
    -   Automatically updates when a quest is turned in.
-   **Guild Bank Tracker:**
    -   Logs gold deposits made to the guild bank.
    -   Provides a UI to view contribution history.
    -   Automatically logs deposits when you use the guild bank deposit button.
-   **Market Telemetry:**
    -   If the Auctionator addon is present, it will alert you if a looted item is valuable.
-   **Fishing Telemetry:**
    -   Logs the coordinates of your fishing spots..
-   **Target Information:**
    -   Displays information about your current target, including whether it is an elite or world boss.

## Slash Commands

-   `/mstart` - Manually starts the combat tracker.
-   `/mstop` - Manually stops the combat tracker.
-   `/mm` - Toggles the display of the Guild Contribution UI.
-   `/mhud` - Toggles the display of the Quest HUD.

## Installation

1.  Download the latest version of the addon.
2.  Unzip the downloaded file.
3.  Copy the `MidnightMarauders` folder into your `World of Warcraft\_retail_\Interface\AddOns` folder.
4.  Restart World of Warcraft.
